Build all necessary jars
========================
in chombo
mvn clean install
sbt publishLocal

in chombo/spark
sbt clean package
sbt publishLocal

in beymani
mvn clean install
sbt publishLocal

in beymani/spark
sbt clean package

Build uber jar
==============
ant -f beymani_spark.xml

uber jar file name is uber-beymani-spark-1.0.jar
